# Primitive
This page contains primitive information

## Types
Here are c++ basic types and their typical size. (Compiler dependent)

| Type               | Bytes  |
|:-------------------|:-------|
| char               | 1      |
| short              | 2      |
| int                | 4      |
| long long          | 8      |
| unsigned char      | 1      |
| unsigned short     | 2      |
| unsigned int       | 4      |
| unsigned long long | 8      |
| float              | 4      |
| double             | 8      |
| bool               | 1 or 4 |
